User Guide

This section is designed to get you familiar with the functions of the PS5 DualSense Controller as well as provide a general overview of how to operate the device. Please refer to the following diagram for the next section of the guide.

A / D / G / H / N:

  • Used for gameplay and menu navigation
  • Joysticks (N, H) control movement
  • Right buttons (G) handle interaction

B (Create Button): Pressing this button opens a menu that allows you to screenshot, record, and stream your current game play.

C (Light Bar):

  • No light: the controller is turned off.
  • Solid white light: the controller is on with no errors.
  • Solid orange light: controller is in rest mode.
  • Flashing blue or white: error connecting or booting.

E (Player Indicator):Lights that glow sequentially depending on the number of controllers currently connected to the PS5 console (up to four). This function allows for local multiplayer for supporting games.

F (Options Button):

  • Pressed while on Home Screen: open the Control Center, allowing you quick access to your options menu or other applications
  • Pressed while a game is running: the game will open its own personal options menu, allowing you to customize the game to your preferences.

I (Speakers): Built-in speaker designed for supplemental and immersive sound effects.

J (PS Button):

  • Single Press: Opens Quick Menu.
  • Press & Hold: Returns to System Home.

K (Headset Jack): Supports any 3.5mm headset plug and changes the output of the audio from the speakers to the headset. Wireless headsets may be connected through the System Menu.

L/M: Area L contains the Microphone. This allows for voice chat without the need for a headset. Pressing the Mute Button (area M) toggles the microphone on and off.

A/B/D/E:These four areas contain more buttons to be used during gameplay. The exact functions of these buttons vary from game to game.

C (USB-C Port):Connecting this port via a USB-C cable to any power source will recharge the controller.

Two additional features are included on the inside of this controller; the Bluetooth 5.1 Wireless Adapter and the Voice Coil Haptic Feedback.

Bluetooth Adapter:This new addition provides an upgrade to traditional infrared signal projectors. Bluetooth 5.1 provides a lower latency ensuring your inputs are spot on.

Haptic Feedback:The DualSense controller contains a set of Voice Coil actuators that can vibrate your controller while playing, to provide that extra depth of immersion. Using Voice Coils instead of traditional rumble motors can fine tune the vibration depending on what is happening in the game (think the difference between rain falling versus driving a car on muddy terrain).
